# 📌 8. CNN MODEL ARCHITECTURE (CORE PART)

---

```python
Conv → Pool → Conv → Pool → Flatten → Dense → Output
```

---

## 🔹 Input Layer

```python
Input(shape=(64,64,3))
```

👉 RGB image

---

## 🔹 Convolution Layer

```python
Conv2D(32, (3,3), relu)
```

👉 Detects:

* Edges
* Patterns

---

## 🔹 Pooling Layer

```python
MaxPooling2D(2,2)
```

👉 Reduces size
👉 Keeps important features

---

## 🔹 Second Conv Layer

```python
Conv2D(64, (3,3))
```

👉 Detects complex patterns

---

## 🔹 Flatten

```python
Flatten()
```

👉 Converts 2D → 1D

---

## 🔹 Dense Layer

```python
Dense(128, relu)
```

👉 Fully connected layer

---

## 🔹 Dropout

```python
Dropout(0.5)
```

👉 Prevents overfitting

---

## 🔹 Output Layer

```python
Dense(num_classes, softmax)
```

👉 Multiclass classification

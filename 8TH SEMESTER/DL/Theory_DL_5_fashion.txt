# 📌 5. CNN MODEL ARCHITECTURE

---

```python
Conv → Pool → Conv → Pool → Flatten → Dense → Output
```

---

## 🔹 Input Layer

```python
Input(shape=(28,28,1))
```

---

## 🔹 Convolution Layer

```python
Conv2D(32, (3,3), relu)
```

👉 Detect edges

---

## 🔹 Pooling

```python
MaxPooling2D(2,2)
```

👉 Reduce size

---

## 🔹 Second Conv Layer

```python
Conv2D(64, (3,3))
```

👉 Detect complex features

---

## 🔹 Flatten

```python
Flatten()
```

👉 Convert 2D → 1D

---

## 🔹 Dense Layer

```python
Dense(128, relu)
```

---

## 🔹 Output Layer


## 🔹 Input

```cpp

```

👉 User enters array size

---

## 🔹 GPU Memory Allocation

```cpp
cudaMalloc(&d_arr, n * sizeof(int));
```

👉 Allocates memory on GPU

---

## 🔹 Copy Data CPU → GPU

```cpp
cudaMemcpy(d_arr, arr, ..., cudaMemcpyHostToDevice);
```

---

## 🔹 Kernel Launch

```cpp
reduce<<<1,1>>>(...)
```

👉 Launch configuration:

| Parameter | Meaning |
| --------- | ------- |
| 1         | blocks  |
| 1         | threads |

👉 Only 1 thread → sequential behavior

---

## 🔹 Copy Back GPU → CPU

```cpp
cudaMemcpy(&h_sum, d_sum, ..., cudaMemcpyDeviceToHost);
```

---

## 🔹 Average

```cpp
float avg = (float)h_sum / n;
```

Sum, Min, Max, Average

# 📌 7. ALGORITHM (WRITE IN EXAM)

---

### 🔹 Parallel Reduction Algorithm

1. Copy input array to GPU
2. Launch kernel
3. Each thread processes data
4. Compute:

   * Sum
   * Min
   * Max
5. Copy results back
6. Compute average

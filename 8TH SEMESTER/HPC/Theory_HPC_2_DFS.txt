# 📌 6. ALGORITHM (WRITE IN EXAM)

### 🔹 Parallel DFS Algorithm

1. Initialize visited array
2. Push start node into stack
3. While stack not empty:

   * Pop node
   * If not visited:

     * Mark visited
     * Print node
     * Parallel explore neighbors
     * Store in local lists
     * Merge results
     * Push into stack

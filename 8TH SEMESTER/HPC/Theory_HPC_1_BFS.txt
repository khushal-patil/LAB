# 📌 6. ALGORITHM (WRITE THIS IN EXAM)

### 🔹 Parallel BFS Algorithm

1. Initialize visited array
2. Insert start node into frontier
3. While frontier not empty:

   * Print nodes
   * Create next_frontier
   * Parallel for each node:

     * Visit neighbors
     * Mark visited (critical section)
     * Add to local list
   * Merge all local lists
   * Update frontier

## 🔹 Q1: What is BFS?

👉 BFS is a traversal algorithm that visits nodes level by level using a queue.

## 🔹 Q2: Difference BFS vs DFS?

| BFS           | DFS            |
| ------------- | -------------- |
| Uses queue    | Uses stack     |
| Level-wise    | Depth-wise     |
| Shortest path | Not guaranteed |
